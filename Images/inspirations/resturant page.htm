Kuliner - Restaurant Landing Page

  • Details
  • Discussion
Live preview

Kuliner is a clean and parallax design food landing page that built-in professional responsive template, elegant style, smooth-animation and have a slew of features.

You can read the online documentation » http://affapress.com/templates/kuliner_responsive_restaurant_landing_page/docs/

Features

  • Built with Bootstrap 3.3.7
  • Fully responsive design
  • Use 1170 grid system
  • Premium design
  • 6 color options
  • 4 custom header layouts
  • Complete PSD & Mockup files
  • Smooth page scrolling
  • Parallax background
  • Subscribe form element with AJAX (use Mailchimp integration)
  • Reservation form element with AJAX
  • Clean template and code
  • Built-in with FontAwesome font pack
  • Cross browser also in IE8
  • HTML5 validation with w3.org
  • Bootstrap validation with bootlint.com
  • Javascript validation with jshint.com
  • SEO friendly
  • Well documented
